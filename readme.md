# Alpha-Edtech-Desafio-HTML/CSS
Repositório para o desafio de HTML/CSS da Alpha Edtech

Este desafio consiste em criar meu próprio portfólio online para exibir meus projetos e habilidades.

Ferramentas utilizadas:
-Lovable.dev: Design de base inicial, com export para HTML/CSS e upload para Figma.
-Figma: Utilizado para desenvolvimento do projeto da página.
-VSCode: Editor utilizado para criação do código, edição dos arquivos, e conexão com Github
-Github: Repositório remoto para upload do projeto, e hosting na Github Pages.
